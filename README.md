## Requerimientos

Utilizar yarn para instalar dependencias

```
yarn install
```

Variables de Entorno:

```
"PORT": "A_PORT"
"WEATHER_KEY": "YOUR_OPENWEATHER_APIKEY"
```
Necesitaras una apikey de OpenWeatherMap.

Si no declaras un puerto, por defecto, sera el puerto 3000
